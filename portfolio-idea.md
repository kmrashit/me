Phase 1: The "Knowledge Nebula" (The Rocket Journey) Instead of manually placing every star, we will treat your skills as a dataset. The Concept: A rocket flies through an infinite field of particles. Each particle is a "knowledge node" (e.g., Redis, Microservices, Event Loop, Financial Modeling). The Backend Role: Your Node.js server holds a JSON file or database of all your concepts. It serves this data to the frontend. The Visuals: Use Three.js to create a "particle system." As the rocket moves, it passes through clusters. One cluster might be "Node.js Core," another "System Design," and another "MBA Finance." When the rocket gets close to a "star," the text label fades in. This creates that "overwhelming" effect of seeing the sheer volume of your expertise.
